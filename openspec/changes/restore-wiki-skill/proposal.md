# Change: Restore wiki generation skill

## Why

The `remove-wiki-skill` change deleted the `gitnexus-wiki` Claude Code skill and all its
registration points. This change reverts that removal — restoring the skill source file,
re-registering it in `setup.ts` and `ai-context.ts`, and re-adding the skill row to the
generated `CLAUDE.md` / `AGENTS.md` skill table.

The `add-claude-subprocess-wiki` proposal (standalone CLI wiki) is unaffected.

## What Changes

- **ADDED** `gitnexus/skills/gitnexus-wiki.md` — skill source file (content from installed artifact)
- **MODIFIED** `gitnexus/src/cli/setup.ts` — add `'gitnexus-wiki'` back to `SKILL_NAMES`
- **MODIFIED** `gitnexus/src/cli/ai-context.ts` — add wiki skill entry back to `skills[]` array
  and add wiki row back to `generateGitNexusContent()` skill table

## Out of Scope

- The standalone CLI wiki (`add-claude-subprocess-wiki`) is untouched.
- `.claude/skills/gitnexus/gitnexus-wiki/SKILL.md` already exists on disk (the
  `remove-wiki-skill` task 4.1 was never executed); it will be regenerated by `gitnexus setup`.

## Impact

- Affected specs: `wiki-skill` (ADDED)
- Affected code/files:
  - `gitnexus/skills/gitnexus-wiki.md` — re-created
  - `gitnexus/src/cli/setup.ts:235` — `SKILL_NAMES` array
  - `gitnexus/src/cli/ai-context.ts:62` — `generateGitNexusContent()` skill table
  - `gitnexus/src/cli/ai-context.ts:151` — `skills[]` array
